# 🚀 Week 1 - Day 2 SQL Query Practice

## 📌 What I Learned Today

Today I practiced SQL query writing using an employee dataset in DB Fiddle as part of my Data Engineering learning journey.

The practice focused on:

* Writing SELECT queries
* Filtering records using WHERE clause
* Using comparison operators
* Applying logical operators
* Working with aggregate functions
* Grouping data using GROUP BY
* Filtering grouped data using HAVING
* Sorting and limiting results
* Using DISTINCT values
* Pattern matching with LIKE
* Using IN, NOT IN, BETWEEN operators
