# 🚀 Week 4 - Day 3 Data Engineering Scenarios

## 📌 Project - 3 Overview

This project simulates common Data Engineering scenarios encountered in ETL workflows and data processing pipelines.

The objective is to clean, transform, optimize, and analyze employee data using PySpark DataFrames.

All tasks were executed using Databricks Workspace and PySpark.

---

# 📚 Project Objectives

* Data Cleaning
* Duplicate Handling
* Missing Value Management
* Data Standardization
* Dataset Joins
* Aggregations
* Performance Optimization
* Business Reporting

---

# 🛠 Project Tasks

1. Remove Duplicate Records
2. Handle Missing Values
3. Standardize Department Names
4. Join Employee and Department Data
5. Department-wise Employee Summary
6. Average Salary by Department
7. Salary Categorization
8. Data Optimization using Repartition
9. DataFrame Caching
10. Final Business Summary Report
