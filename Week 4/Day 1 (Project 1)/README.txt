# 🚀 Week 4 - Day 1 Employee Salary Analytics

## 📌 Project - 1 Overview

This project focuses on analyzing employee salary data using PySpark DataFrame operations and Window Functions.

The objective is to generate meaningful business insights such as salary distribution, department-wise analytics, employee rankings, and salary trend reports.

All tasks were executed using Databricks Workspace and PySpark.

---

# 📚 Project Objectives

* Analyze employee salaries
* Generate department-wise reports
* Generate city-wise salary analytics
* Categorize employees into salary bands
* Identify top and bottom earners
* Create business-ready salary reports

---

# 🛠 Project Tasks

1. Top 3 Highest Salaries Department-wise
2. Average Salary City-wise
3. Employee Salary Bands
4. Yearly Salary Report
5. Employees Earning Above Department Average
6. Highest Paid Employee per Department
7. Lowest Paid Employee per Department
8. Department-wise Salary Summary
9. Department-wise Average Salary
10. Department-wise Employee Count
