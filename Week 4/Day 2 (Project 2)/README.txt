# 🚀 Week 4 - Day 2 HR Dashboard

## 📌 Project - 2 Overview

This project focuses on Human Resource Analytics using PySpark DataFrames.

The objective is to generate HR reports and business insights such as employee demographics, hiring trends, salary analytics, and workforce distribution.

All tasks were implemented using Databricks and PySpark.

---

# 📚 Project Objectives

* Analyze employee demographics
* Generate department-wise HR reports
* Analyze hiring trends
* Analyze salary distribution
* Create dashboard-ready summary reports

---

# 🛠 Project Tasks

1. Count Male and Female Employees
2. Average Age Department-wise
3. Hiring Trends by Year
4. City-wise Employee Distribution
5. Highest Paid Employees
6. Lowest Paid Employees
7. Department-wise Employee Count
8. Gender-wise Average Salary
9. Employees Above Average Salary
10. HR Summary Dashboard
